# AutoStyle — Интернет-витрина автоаксессуаров

Современный многостраничный веб-сайт интернет-витрины автомобильных аксессуаров с адаптивной версткой, выполненный в стильной темной теме с яркими оранжевыми акцентами.

## 🚀 Основной функционал и особенности проекта

*   **Многостраничная структура:** Сайт состоит из 5 полноценных взаимосвязанных страниц (Главная, Каталог, Карточка товара, О нас, Контакты).
*   **Умная система заказа (Прямая покупка):** Вместо сложной корзины реализован удобный флоу для витрины — при клике на кнопку «Купить» в каталоге пользователь перенаправляется на страницу контактов, а поле обратной связи автоматически заполняется текстом выбранного товара.
*   **Полная адаптивность:** Интерфейс оптимизирован под два ключевых разрешения — Desktop (1440px) и Mobile (360px).
*   **Интерактивное мобильное меню:** Для мобильной версии реализовано всплывающее меню-гамбургер (Тонирование/скрытие по клику).
*   **Удобная навигация:** На всех внутренних страницах настроены «хлебные крошки» для быстрой ориентации пользователя.
*   **Валидация форм:** Форма обратной связи на странице контактов проверяется встроенными средствами HTML5 и скриптом JavaScript перед отправкой.

## 🛠 Технологический стек

*   **HTML5** — семантическая разметка страниц.
*   **CSS3** — адаптивная сетка (Grid Layout, Flexbox), использование CSS-переменных (`:root`) для легкого управления цветовой палитрой и стилями.
*   **JavaScript (Vanilla JS)** — логика мобильного меню, валидация формы и динамическое заполнение полей через параметры URL (`URLSearchParams`).
